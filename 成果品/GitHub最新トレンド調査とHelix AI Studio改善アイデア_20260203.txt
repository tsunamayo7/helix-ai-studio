【2.6 Ollama ストリーミング実装 (2026)】
-------------------
Ollama Python ライブラリによるストリーミング実装:

基本パターン (stream=True):
```python
from ollama import chat
stream = chat(
    model='gemma3',
    messages=[{'role': 'user', 'content': 'Why is the sky blue?'}],
    stream=True,
)
for chunk in stream:
    print(chunk['message']['content'], end='', flush=True)
```

Clientクラス利用:
```python
from ollama import Client
client = Client(host='http://localhost:11434')
for part in client.chat('gemma3', messages=messages, stream=True):
    print(part.message.content, end='', flush=True)
```

非同期ストリーミング:
```python
from ollama import AsyncClient
async for part in await AsyncClient().chat(model='gemma3', messages=[message], stream=True):
    print(part['message']['content'], end='', flush=True)
```

2026新機能:
- ツール呼出しとストリーミングの同時対応
- リアルタイムツール呼出しチェック (chunk.message.tool_calls)

参考:
- Ollama Python library: https://github.com/ollama/ollama-python
- Streaming responses with tool calling: https://ollama.com/blog/streaming-tool
- Streaming - Ollama: https://docs.ollama.com/capabilities/streaming

```python
import qdarktheme
app = QApplication(sys.argv)
```

OS同期:

```python
app.setStyleSheet(qdarktheme.load_stylesheet())
```
