# Screenshots

This directory contains screenshots for the README.

## File Naming Convention

Use descriptive names with timestamps:
- `HelixAIStudio_YYYYMMDD_HHMMSS.png`

## Recommended Screenshots

1. **mixAI Chat** - Main chat interface with 3-phase pipeline visualization
2. **mixAI Settings** - Configuration panel for Phase 2 models and retry settings
3. **soloAI Chat** - Direct Claude CLI chat interface
4. **General Settings** - Claude model selection and MCP server management
5. **BIBLE Panel** - BIBLE detection and management UI
6. **Neural Flow Visualizer** - Real-time phase execution visualization
7. **VRAM Budget Simulator** - Memory budget planning tool

## How to Add Screenshots

1. Take screenshots of the application
2. Save them to this directory with descriptive names
3. Update `../README.md` and `../README_ja.md` image paths
4. Commit and push

Example markdown for README:

```markdown
![mixAI Chat](docs/screenshots/mixai_chat.png)
```

## Image Guidelines

- **Format**: PNG (preferred) or JPEG
- **Resolution**: 1920x1080 or higher
- **Content**: Hide any sensitive information (API keys, file paths)
- **Size**: Keep under 500KB per image (use compression if needed)
